About Me

Musician, Web Developer

Hey there! My name is Emma. After completing a bachelor's degree in Clarinet Performance, I'm transitioning into a career in Web Development. I'm currently working on an Associate's degree in Interface Design and Web Development and looking for experience so that I can land an entry-level web developer position.

I focus on accessibility and usability in my web creations. I don't want to bar anyone from using a website of mine. While working hard developing, I'm beginning to also enjoy web design as I get more comfortable using Figma. I'm even starting to make my own vector images. Right now I have the most experience using HTML, CSS, and JavaScript without any frameworks at the moment, although I have dabbled in Svelte and React. As I continue coding and learning, I want to keep contributing to open-source projects and start to work on larger projects with others.
